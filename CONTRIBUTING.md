# Contributing to AgentLens

First off, thank you for considering contributing to `AgentLens`! It's people like you that make the security ecosystem safer for everyone.

## Getting Started

1. Fork the repository on GitHub.
2. Clone your fork locally.
3. Setup a virtual environment (`python -m venv .venv` and `source .venv/bin/activate`).
4. Install the package in editable mode (`pip install -e .`).

## Submitting a Pull Request

1. Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
2. Make your targeted code changes.
3. Commit your changes (`git commit -m 'Add amazing feature'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request!

## Reporting Issues

If you find a bug or have a suggestion, please open an issue in the GitHub issue tracker.
Ensure you provide clear reproduction steps if you are reporting a codebase bug or a false positive inside the static analyzer.
